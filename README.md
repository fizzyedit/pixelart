Hello!

These are the `.fiz` files for all the pixel art I've worked with since starting to develop the application. Most artwork here is drawn by me, but there are some kenny assets I imported just for testing, or use in another game project.

To open these files, please install `fizzy` from [here](https://fizzyed.it), download this repository, and then open the folder through the UI in fizzy. You will then be able to access the artwork and animations in these files.
